`@kortix/sdk` is the typed client for the Kortix platform. It wraps the Kortix
REST API and the OpenCode runtime in one interface. Your app never calls the
raw API or the runtime directly.

## Install

```bash
npm install @kortix/sdk
```

## A first call

```ts
import { createKortix } from '@kortix/sdk';

const kortix = createKortix({
  backendUrl: 'https://api.kortix.com/v1',
  getToken: async () => process.env.KORTIX_API_KEY!,
});

const session = kortix.session(projectId, sessionId);
await session.ensureReady();
await session.send('Refactor the auth module');
```

`createKortix` returns one client. `getToken` supplies your bearer token — a
`kortix_pat_...` API key or a Supabase JWT. `ensureReady` starts or resumes
the session sandbox and waits until it accepts work.

`createKortix` gives you an imperative client: call methods for every action,
like projects, sessions, secrets, and triggers. `@kortix/sdk/react` gives you
hooks for live UI data — `useSession` runs a whole session in one hook.
