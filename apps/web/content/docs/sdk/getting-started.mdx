This page installs `@kortix/sdk`, creates a client, and sends your first message to a session.

## Install

```sh
pnpm add @kortix/sdk
```

The core client runs in Node 18+, Bun, browsers, and edge runtimes such as Cloudflare Workers. `react` (18+) and `@tanstack/react-query` (5.75+) are optional peers, needed only for the [React hooks](/docs/sdk/react).

## Create a client

Call `createKortix` once, with your API base URL and a function that returns your token.

```ts
import { createKortix } from '@kortix/sdk';

export const kortix = createKortix({
  backendUrl: 'https://api.kortix.com/v1',
  getToken: async () => process.env.KORTIX_API_KEY!,
});
```

`backendUrl` and `getToken` are the only required fields. The SDK calls `getToken` on every request and caches nothing — your host owns token storage and refresh.

Do not import `@opencode-ai/sdk` directly. The SDK re-exports the parts you need.

## Authenticate

Create an API key at **User settings → API keys**. Choose a scope:

- **Account** — reaches every project in the account. This is the default.
- **Project** — reaches one project. Requests to any other project return `403`.

Copy the key. It starts with `kortix_pat_` and shows only once. Store it as a secret. Return it from `getToken`.

See [Authentication](/docs/sdk/auth) for token types and the service account gotcha.

## Create and start a session

A session is one agent run, in its own sandbox on its own branch.

```ts
const created = await kortix.project(projectId).sessions.create();
const session = kortix.session(projectId, created.session_id);
await session.ensureReady();
```

`ensureReady()` starts the sandbox and waits until it answers. If the sandbox is still starting, it throws an `ApiError` with `code: 'RUNTIME_UNAVAILABLE'`. Retry after a short delay. See [Error handling](/docs/sdk/error-handling).

## Send a message

```ts
await session.send('Add a README');
```

`send()` calls `ensureReady()` for you, then sends the message to the agent. To read the agent's reply as it streams in, see [Streaming](/docs/sdk/streaming).
