New kortix gateway CLI and tightened compliance controls

### New
- **`kortix gateway`** — configure and inspect a project's LLM gateway straight from the CLI: default-model routing and fallback chains, spend budgets, external gateway API keys, usage and request logs, and a built-in model playground. Complements `kortix providers` (which connects the credentials) so the whole gateway can be driven programmatically.

### Improved
- **Compliance** — all-region AWS security controls are now codified, so coverage stays consistent across every region instead of drifting.
