# Variant Sets

Use this when the user wants multiple directions. Match their requested count when they give one.

```md
## Direction
[One sentence — the shared goal]

## Variants

### [Name]
Closest to the request. Strong fidelity, minimal interpretation.

### [Name]
More stylized or exploratory. Core subject and constraints intact.

### [Name]
Most reliable for the target format, crop, and model.

## Avoid
[Shared constraints only]

## Model
[Model + one-line reason, if relevant]
```

## Rules

- Every variant must differ on a real creative axis: composition, lighting, material, environment, color, camera distance, render style, or level of abstraction. Swapping adjectives is not a variant.
- For mixed crops, transparency needs, or mixed reference roles, give each variant its own model + aspect-ratio note instead of a single shared model line.
- For 4+ variants, name explicit axes rather than padding generic slots — e.g. "Tight product crop," "Wide editorial environment," "High-key studio," "Low-light cinematic," "Graphic color-blocked."
