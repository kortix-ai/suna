# Hooks & Post Templates

The first line decides whether the rest is read. Pick a hook type to the goal, then fill a skeleton. Keep every post to one idea and one action.

## Hook formulas

**Curiosity** (best for reach)
- "I was wrong about [common belief]."
- "The real reason [outcome] happens isn't what you think."
- "[Impressive result] — and it only took [surprisingly short time]."

**Story** (best for watch time / dwell)
- "Last week, [unexpected thing] happened."
- "I almost [big mistake]."
- "[N] years ago I [past state]. Today, [current state]."

**Value** (best for saves)
- "How to [outcome] (without [common pain]):"
- "[Number] [things] that [outcome]:"
- "Stop [common mistake]. Do this instead:"

**Contrarian** (best for comments)
- "Unpopular opinion: [bold statement]"
- "[Common advice] is wrong. Here's why:"
- "I stopped [common practice] and [positive result]."

## LinkedIn skeletons

**Story post**
```
[Hook: unexpected outcome or lesson]
[Set the scene]
[The challenge]
[What you tried / the turning point]
[The result]
[The lesson for the reader]
[Question to prompt comments]
```

**Contrarian take**
```
[Unpopular opinion, stated flat]
Here's why:
[Reason 1]
[Reason 2]
[Reason 3]
[What to do instead]
[Invite debate: "Am I wrong?"]
```

**List post**
```
[N] things I learned about [topic] after [credibility builder]:
1. [Point] — [one line]
2. [Point] — [one line]
3. [Point] — [one line]
[Wrap-up insight]
[Which resonates most?]
```

## Twitter/X skeletons

**Tutorial thread**
```
1/ [Hook + promise: "Here's exactly how to [outcome], step by step:"]
2–6/ [One step per tweet, with the detail]
last/ [Recap + CTA: "Follow for more on [topic]"]
```

**Story thread**
```
1/ [Intriguing hook — the outcome or stakes]
2/ [Setup]
3–5/ [Conflict and turns, one beat per tweet]
6/ [Resolution + lesson]
last/ [CTA]
```

## Carousel skeleton (LinkedIn / Instagram)

```
Slide 1  Hook — promise a payoff, make them swipe
Slide 2  The problem / why it matters
Slide 3–N  One step or point per slide
Slide N+1  Recap
Last     CTA — save, follow, or comment
```
Hand the visual build to `kortix-image`; write the slide copy here.

## CTA menu

Question ("What would you add?") · Agreement ("Agree or disagree?") · Share ("Tag someone who needs this") · Save ("Save this for later") · Follow ("More on [topic] — follow"). One per post.
