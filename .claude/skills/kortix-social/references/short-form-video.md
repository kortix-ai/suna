# Short-Form Video: Hooks, Structures & Scripts

For TikTok, Instagram Reels, and YouTube Shorts. All 9:16, 15–60s. The script is the deliverable; hand any cover frame to `kortix-image`.

## The 3-second rule

You have 3 seconds to stop the scroll. Land all three hooks in the first second:

```
[VISUAL HOOK] + [VERBAL HOOK] + [TEXT OVERLAY]
```

Show the payoff or the problem immediately — never build up to it.

## Hook library

**Curiosity** — "The secret to [outcome] nobody talks about" · "I tried [thing] for 30 days and was NOT expecting this" · "Why does nobody talk about this?"

**Value** — "How to [outcome] in [timeframe]" · "[N] [things] that will [benefit]" · "Stop doing [practice] — here's why"

**Story** — "3 months ago I [bad state]. Today I [good state]." · "Here's why I stopped [practice]" · "So this just happened…"

**Contrarian** — "Unpopular opinion: [bold statement]" · "[Common advice] is actually wrong" · "Most [audience] get this completely wrong"

## Structures

**Problem → Solution (15–30s)**
```
0–3s    Hook: state the problem
3–10s   Agitate: why it matters
10–25s  Solution: the method / tip / product
25–30s  CTA
```

**List (30–60s)**
```
0–3s    Hook: "[N] things that [outcome]"
3–50s   One item every 5–8s
50–60s  CTA
```

**Tutorial (30–60s)**
```
0–3s    Hook: show the end result first
3–8s    "Here's how…"
8–50s   Quick, clear steps
50–60s  Result + CTA
```

## Scripting template

```markdown
## Video: [Working title]
Platform: TikTok / Reels / Shorts   Length: XXs   Format: [talking head / demo / screen-rec / slideshow]

### Hook (0–3s)
- Visual: [what they see]
- Audio: [what they hear]
- Text overlay: [on-screen text]

### Body (3–Xs)
- [timestamp] — [beat]
- [timestamp] — [beat]

### CTA (final 3–5s)
- Verbal / Text / Action: [follow, comment, link in bio…]

### Production notes
- Sound: [trending audio / VO]
- B-roll: [clips needed]
- Graphics: [overlays / captions]
```

## Captions

Most short-form is watched on mute; subtitles lift watch time 25–40%.
- Max 2 lines on screen, 3–5 words per line
- Bold sans-serif with a black outline; highlight the key word in a second color
- Time to speech exactly
- Tools: CapCut (free), Descript, Captions.ai

## Common mistakes

Slow hook · no text overlay · bad audio (kills retention instantly) · too long · no CTA · ignoring comments in the first hour.
