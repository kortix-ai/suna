# Platform Limits

Character counts, hashtag limits, and "see more" thresholds. Verify against current platform docs before a high-stakes launch — these shift.

| Platform | Max chars | Visible before "more" | Recommended hashtags | Hard hashtag cap |
| --- | --- | --- | --- | --- |
| LinkedIn | 3,000 | ~210 | 3–5 | none |
| Twitter/X | 280 (25,000 Premium+) | full 280 | 1–2 | none |
| Instagram | 2,200 caption | ~125 | 3–5 | 5 |
| TikTok | 4,000 caption | ~150 | 3–5 | 5 (since Aug 2025) |
| Facebook | 63,206 | ~40–80 ideal | 1–2 | none |
| YouTube | 100 title / 5,000 desc | ~70 title, ~100 desc | 3–5 | 15 (exceeding makes YouTube ignore *all*) |
| Threads | 500 (+10,000 expandable) | first 1–2 lines | 1 | 1 topic tag |
| Pinterest | 100 title / 500 desc | ~50 (desktop) | 2–5 | 20 |

## Notes

- **YouTube** — first 3 description hashtags render above the title as links. Shorts: 1–5 hashtags.
- **Threads** — one topic tag per post; the platform isn't hashtag-driven.
- **Pinterest** — hashtags deprioritized; write keywords as natural sentences for SEO instead.
- **LinkedIn / X** — keep links out of the post body (reach penalty); use first comment or a reply.
